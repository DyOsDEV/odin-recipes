PROJECT DESCRIPTION

Odin Recipes is a project to practice HTML skills by creating a basic recipe website.

PREREQUISITES

- Git
- Web Browser.

USAGE

- Explore the index page to find different recipes.
- Click on the links to view details for each recipe.
